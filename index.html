Chess Live Ratings

Github

Built with SwiftUI, this scrapes data from 2700chess.com using the SwiftSoup library to accurately display information about the top ranked chess players with categories such as Classical Rating, Rapid Rating, Blitz Rating, and Junior Rating. This supports light an dark mode.

Rice Prices

Github

Using WidgetKit and SwiftUI, this scrapes data from MarketBusinessInsider every 15 minutes to display the price and changes in price of 100lbs of rice using SwiftSoup. This widget supports light mode and dark mode.
